This repository contains **basic Java practice problems** designed to help beginners **improve their programming and logical thinking skills**.

The goal of this repository is to provide **simple and clear Java solutions** for common beginner problems such as:

* Conditional statements
* Loops
* Pattern problems
* Basic logic building exercises

In the future, more problems will be added including **math problems and logical reasoning problems**.

This repository can be used by **students, beginners, and anyone learning Java**.

---

# 📚 Topics Covered

### 🔹 Conditional Statement Problems

Examples using:

* `if`
* `if-else`
* `else-if`
* `switch`

These problems help understand **decision making in Java**.

---

### 🔹 Loop Problems

Practice problems using:

* `for loop`
* `while loop`
* `do-while loop`

These help in understanding **iteration and repetition in programs**.

---

### 🔹 Pattern Problems

This repository includes **20 different pattern problems** such as:

* Star patterns
* Number patterns
* Pyramid patterns
* Reverse patterns

Pattern problems are very useful for **improving logic and loop understanding**.

Example:

```
*
**
***
****
*****
```
